# ⚡ SATHANIC STRIKE - Realistic 2D Shooter

Mini Militia style-il ulla oru fast-paced, offline action game aanu **SATHANIC STRIKE**. Ithu poornamayum GitHub Pages vazhi browser-il kalikkan sadhikum. Dark neon theme-um realistic motion effects-um ee game-ine kooduthal interesting akkunu.

## 🎮 Game Features
- **Realistic Motion Blur:** Kalikkumpol nalla smooth experience nalkunnu.
- **Neon Graphics:** Modern high-graphics look.
- **Offline Play:** Internet illatheyum time-pass ayi kalikkaam.
- **Auto-Spawn Enemies:** Enemy rounds koodunthorum game kooduthal challenging aakum.

## 🕹️ Engane Kalikkam? (How to Play)
- **Movement:** Game window-il click cheyyunnathinu anusarichu player control cheyyam.
- **Shooting:** Mouse/Touch upayogichu enemy-kku nere shoot cheyyuka.
- **Objective:** Kooduthal enemies-ne nashihippichu high score build cheyyuka.

## 🛠️ Tech Stack
- **HTML5 Canvas:** Game rendering-nu vendi.
- **CSS3:** Realistic lighting effects and UI.
- **JavaScript:** Game engine and physics logic.
- **GitHub Pages:** For hosting and live access.

## 🚀 Installation & Setup
Ithu ningalude swantham GitHub-il setup cheyyan:
1. Ee repository **Fork** cheyyuka.
2. `Settings > Pages` option-il poyi `Main Branch` select cheyyuka.
3. Labhikkunna URL-il click cheythu game kalikkuka.
